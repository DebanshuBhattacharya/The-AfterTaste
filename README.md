# 🍽️ AfterTaste

**AfterTaste** is a modern and stylish food blog website built using **HTML, CSS, and JavaScript**. It is designed to showcase delicious dishes, recipes, and food stories with an engaging user experience and smooth animations.

## ✨ Features

* 🍕 Beautiful modern UI with responsive design
* 🍔 Food cards with images and categories
* 🔍 Search and filter functionality
* 🎬 Scroll reveal animations
* 📱 Mobile-friendly layout
* ⚡ Smooth scrolling navigation
* ❤️ Clean and attractive design

## 🛠️ Built With

* **HTML5** – Structure
* **CSS3** – Styling & Animations
* **JavaScript** – Interactivity

## 📂 Project Structure

```bash
AfterTaste/
│── index.html
│── style.css
│── script.js
│── assets/
```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/aftertaste.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
